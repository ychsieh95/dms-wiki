# 屬性相剋表

作者：Mooncake aka 月餅

前言：屬性 Counter 表，如果遇到「沒有任何弱點的遠征王」，三轉未學聖靈宜開雷、四轉宜開聖靈屬，**所有的遠征王有調整成弱雷、弱毒**。未來有機會有圖片版的。
