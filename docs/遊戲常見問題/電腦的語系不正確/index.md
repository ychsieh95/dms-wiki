# 電腦的語系不正確

1. 開啟控制台。

2. 選擇 `Clock and Region`。

    ![clock-and-region](0.png)

3. 選擇 `Region`。

    ![region](1.png)

4. 開啟 `Administrative` 分頁，並按下 `Change system locale`。

    ![administrative](2.png)

5. 選擇 `Chinese (Tranditional , Taiwan)`，並按下 `OK`。

    ![change-system-locale](3.png)

6. 重開機後即完成設定。
